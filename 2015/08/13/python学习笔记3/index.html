python学习笔记03

字符串

基本用法类似列表和元组

Output

capitalize()

首字母大写

Output

casefold()

所有字母变小写

Output

center(width[, fillchar])

按照指定长度将字符串居中，默认空白处使用空格，也可以指定填充字符

Output

count(sub[, start[, end]])

返回子字符串在整个字符串中指定位置出现的次数

Output

encode(encoding=”utf-8”, errors=”strict”)

按照encoding指定的编码格式对字符串进行编码，这个暂时不是很清楚，跳过

Output

endswith(suffix[, start[, end]])

测试字符串是否以sub串结尾,可以指定位置

Output

expandtabs(tabsize=8)

将字符串中的tab转换成空格,如果不指定默认是8

Output

find(sub[, start[, end]])

检查sub是否在字符串中,有则返回第一次出现的位置,否则返回-1
无论是否给出start end, 返回的都是绝对位置,而不是针对start的相对位置

Output

index(sub[, start[, end]])

与find方法类似,但是如果没找到则会抛出异常

Output

isalnum()

判断字符串是否仅仅包含字母和数组

Output

isalpha()

判断字符串是否仅仅包含字母

Output

isdecimal()

判断字符串是否仅仅包含十进制数

Output

isdigit(), isnumeric()

和isdecimal基本类似都是判断是否是数字.
但是不同的是isnumeric可以判断其他语言的数字
byte object只能用isdigit

Output

islower(), isupper()

islower 判断字符串中包含字母是否全部为小写
isupper 判断字符串中包含字母是否全部为大写

Output

isspace()

字符串是否只包含空格

Output

istitle()

是否所有的单词仅仅首字母大写

Output

join(iterable)

将字符串本身做为间隔插入到iterable中去

Output

ljust(width[, fillchar])

左对齐后面补足空格，保证长度等于width

Output

lower(), upper()

lower 所有字母小写
upper 所有字母大写

Output

lstrip([chars]), lstrip([chars])

lstrip 去除左边所有空格
rstrip 去除右边所有空格

Output

partition(sep)

找到sep并以sep为分隔将字符串变为3个元组

Output

replace(old, new[, count])

字符串替换,并可以指定替换次数

Output

rfind(sub[, start[, end]])

rindex(sub[, start[, end]])

rjust(width[, fillchar])

rpartition(sep)

都是对应函数的右起操作

split(sep=None, maxsplit=-1)

默认以空格来对字符串进行切片处理,切割成列表
也可是指定切割符号,也可是指定切割次数

Output

splitlines([keepends])

以换行符\n来切割字符串列表

Output

startswith(prefix[, start[, end]])

测试字符串是否以prefix开始,可以指定位置

Output

strip([chars])

默认删除字符串前面和后面的空格，但是中间不删除，也是指定删除前后字符

Output

swapcase()

翻转字符串中的大小写

Output

title()

将所有的单词，都进行首字母大写

Output

translate(map)

根据定义好的字符映射表，来替换字符完成翻译
而map则由str.maketrans(‘a’, ‘b’)来定制
注意替换仅仅是字符对字符的替换

Output

zfill(width)

返回长度为width的字符串，原字符串右对齐，前面用0填充

Output

format(args, *kwargs)

Output

format_map(mapping)
